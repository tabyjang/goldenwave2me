# 골든웨이브 마케팅 템플릿 시스템 - 작업 시나리오

## 1. 프로젝트 개요

### 목표
- 5개의 마케팅 에이전시 웹사이트 템플릿 제작
- 포트폴리오로 활용 가능한 데모 사이트
- 고객사별 커스터마이징 가능한 실제 작동 시스템

### 기반 기술 (next-starter-kit)
```
next-starter-kit/
├── Next.js 16.1.1 + React 19
├── Tailwind CSS 4
├── Framer Motion (motion)
├── next-themes (다크모드)
├── next-intl (다국어)
├── shadcn/ui 스타일 컴포넌트
├── Three.js (3D)
├── Storybook 10
├── Playwright E2E
└── NextAuth (인증)
```

## 3. 추천 방식: Option A (단일 프로젝트 + 테마 시스템)

### 이유
1. **포트폴리오 용도**: 단일 사이트에서 5개 템플릿을 한눈에 보여줄 수 있음

3. **유지보수**: 공통 버그 수정 시 한 번만 수정
4. **배포 비용**: Vercel 무료 플랜으로 충분

#### 6.2 동적 설정 로딩

```typescript
// src/lib/config.ts
import { defaultConfig } from '@/config/site.config'

export async function getSiteConfig() {
  const configName = process.env.NEXT_PUBLIC_SITE_CONFIG || 'goldenwave'

  try {
    const config = await import(`@/config/clients/${configName}.config`)
    return config.default
  } catch {
    return defaultConfig
  }
}

export function getTheme() {
  return process.env.NEXT_PUBLIC_THEME || 'minimal'
}
```
