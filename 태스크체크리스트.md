# 골든웨이브 마케팅 홈페이지 제작 태스크 체크리스트

## 프로젝트 개요

**회사명**: 골든웨이브 마케팅 (GoldenWave Marketing)

**주요 서비스**:
- 홈페이지 제작
- 브랜드 블로그 관리대행
- 업무 자동화 솔루션

**타겟 고객**: 중소기업, 스타트업, 자영업자 (법률·세무, 부동산, 금융, 미용, 교육, 요식업, 패션, IT 등)

**브랜드 키워드**: 신뢰, 전문성, 성장, 파도처럼 퍼지는 마케팅 효과

---

## 5가지 템플릿 아이디어

### 템플릿 1: 미니멀 프로페셔널 (Minimal Professional)

**컨셉**: 깔끔하고 신뢰감 있는 기업형 디자인

**특징**:
- [ ] 화이트 배경 + 골드/네이비 포인트 컬러
- [ ] 명확한 그리드 시스템과 여백 활용
- [ ] 산세리프 타이포그래피 (Pretendard, Inter)
- [ ] 직관적인 서비스 카드 레이아웃
- [ ] 신뢰 배지, 고객 로고 롤링
- [ ] 깔끔한 CTA 버튼

**장점**: 빠른 로딩, 높은 가독성, B2B 신뢰감
**단점**: 차별화 부족, 임팩트 약함

**레이아웃**:
```
[헤더: 로고 + 네비게이션 + 상담문의 버튼]
[히어로: 헤드라인 + 서브카피 + CTA]
[서비스 섹션: 3개 서비스 카드]
[포트폴리오: 케이스 스터디 그리드]
[고객후기: 캐러셀 슬라이더]
[CTA 섹션: 무료 상담 신청]
[푸터: 연락처 + SNS + 저작권]
```

---

### 템플릿 2: 대담한 골드러시 (Bold Gold Rush)

**컨셉**: 네온 골드와 대담한 타이포그래피로 강렬한 인상

**특징**:
- [ ] 네온 골드 그라데이션 (#FFD700 → #FFA500)
- [ ] 대담한 키네틱 타이포그래피
- [ ] 비대칭 레이아웃
- [ ] 마우스 인터랙션 효과
- [ ] 굵은 헤드라인 (800+ weight)
- [ ] 파티클/그레인 텍스처

**장점**: 강렬한 첫인상, 트렌디함, 기억에 남음
**단점**: 일부 고객층에 과할 수 있음

**레이아웃**:
```
[풀스크린 히어로: 움직이는 골드 텍스트]
[스크롤 트리거 서비스 소개]
[비대칭 포트폴리오 갤러리]
[숫자 카운터 섹션 (성과 지표)]
[프로세스 타임라인]
[문의 폼: 대담한 CTA]
```

---

### 템플릿 3: 다크 프리미엄 (Dark Premium)

**컨셉**: 고급스러운 다크 테마 + 골드 악센트

**특징**:
- [ ] 다크 배경 (#0a0a0a, #1a1a1a)
- [ ] 골드 악센트 (#D4AF37, #B8860B)
- [ ] 세련된 호버 애니메이션
- [ ] 글라스모피즘 카드 UI
- [ ] 부드러운 스크롤 애니메이션
- [ ] 다크/라이트 모드 토글

**장점**: 프리미엄 이미지, 세련됨, 눈의 피로 감소
**단점**: 콘텐츠 가독성 주의 필요

**레이아웃**:
```
[플로팅 헤더 (스크롤 시 블러 효과)]
[시네마틱 히어로: 배경 영상/파티클]
[글라스 카드 서비스 섹션]
[호버 효과 포트폴리오]
[브랜드 스토리 섹션]
[프리미엄 가격표]
[다크 푸터 + 골드 악센트]
```

---

### 템플릿 4: 인터랙티브 웨이브 (Interactive Wave)

**컨셉**: 파도/물결 모션으로 브랜드 아이덴티티 표현

**특징**:
- [ ] SVG/Canvas 파도 애니메이션
- [ ] 스크롤 기반 스토리텔링
- [ ] 부드러운 곡선 섹션 구분선
- [ ] 패럴랙스 스크롤 효과
- [ ] 마우스 따라다니는 웨이브
- [ ] 블루/골드 그라데이션

**장점**: 브랜드 아이덴티티 강조, 몰입감, 독창성
**단점**: 개발 복잡도 높음, 성능 최적화 필요

**레이아웃**:
```
[웨이브 애니메이션 히어로]
[물결 구분선 + 서비스 소개]
[스크롤 트리거 숫자 애니메이션]
[웨이브 배경 포트폴리오]
[타임라인 프로세스 (물결 경로)]
[플로팅 문의 버튼]
```

---

### 템플릿 5: 원페이지 임팩트 (One-Page Impact)

**컨셉**: 빠르고 강렬한 원페이지 랜딩

**특징**:
- [ ] 단일 페이지 구조
- [ ] 풀스크린 섹션 (100vh)
- [ ] 스냅 스크롤 (Scroll Snap)
- [ ] 미니멀 네비게이션 (도트)
- [ ] 빠른 로딩 (< 2초)
- [ ] 모바일 퍼스트

**장점**: 빠른 로딩, 집중력 높음, 전환율 최적화
**단점**: 정보량 제한, SEO 불리

**레이아웃**:
```
[섹션 1: 풀스크린 히어로 + CTA]
[섹션 2: 서비스 3가지 (가로 슬라이드)]
[섹션 3: 포트폴리오 하이라이트]
[섹션 4: 프로세스 4단계]
[섹션 5: 후기 + 문의폼]
```
