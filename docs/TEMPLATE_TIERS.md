## Starter 티어

### 개요
빠르게 온라인에 존재감을 만들고 싶은 분을 위한 필수 페이지 구성입니다.
개인 포트폴리오, 랜딩 페이지, 소규모 비즈니스 소개 사이트에 적합합니다.

### 가격
**₩39,000** (USD $29)

### 포함 페이지 (5개)

| 페이지 | 설명 | 주요 섹션 |
|--------|------|-----------|
| **Home** | 메인 랜딩 페이지 | 히어로, 기능 소개, 소셜 증명, CTA |
| **About** | 소개 페이지 | 회사/개인 스토리, 미션, 비전 |
| **Contact** | 연락처 페이지 | 문의 폼, 연락처 정보, 지도 |
| **Blog List** | 블로그 목록 | 포스트 그리드, 카테고리 필터 |
| **Blog Post** | 블로그 상세 | MDX 콘텐츠, 작성자 정보, 공유 버튼 |

### 포함 기능

**디자인 & UI**
- 반응형 디자인 (모바일/태블릿/데스크탑)
- 라이트/다크 모드
- Tailwind CSS 스타일링
- shadcn/ui 컴포넌트

**기술 스택**
- Next.js 16 App Router
- TypeScript
- SEO 최적화 (메타태그, sitemap, robots.txt)

**콘텐츠 관리**
- MDX 기반 블로그 시스템
- 다국어 지원 기본 구조 (ko/en)

### 이런 분께 추천
- 개인 포트폴리오를 만들고 싶은 개발자/디자이너
- 간단한 회사 소개 사이트가 필요한 소규모 사업자
- 빠르게 랜딩 페이지를 론칭하고 싶은 스타트업
- 웹 개발을 배우면서 실습하고 싶은 입문자

## 자주 묻는 질문

### Q: 어떤 티어를 선택해야 하나요?

**Starter**: 빠르게 온라인에 사이트를 올리고 싶고, 회원 기능이 필요 없다면
**Pro**: 회원 가입, 로그인이 필요하고 비즈니스 사이트를 만들고 싶다면
**Enterprise**: SaaS 제품이나 대시보드가 필요한 서비스를 만든다면

### Q: 업그레이드가 가능한가요?

네, 차액만 지불하고 상위 티어로 업그레이드할 수 있습니다.
- Starter → Pro: ₩50,000
- Pro → Enterprise: ₩100,000
- Starter → Enterprise: ₩150,000

### Q: 기술 지원은 어떻게 받나요?

- **Starter**: 문서 및 가이드 제공
- **Pro**: 30일 이메일 지원
- **Enterprise**: 90일 이메일 지원 + 1회 화상 미팅

### Q: 라이선스 정책은 어떻게 되나요?

- 1개 라이선스 = 1개 프로젝트 (1개 도메인)
- 클라이언트 프로젝트에 사용 가능 (재판매 불가)
- 소스 코드 수정 자유

### Q: 환불 정책은 어떻게 되나요?

- 구매 후 7일 이내, 다운로드 이력이 없는 경우 100% 환불
- 다운로드 후에는 환불 불가
